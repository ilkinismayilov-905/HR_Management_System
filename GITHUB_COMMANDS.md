 # GITHUB COMMANDS

 ## Merge to Main :
1. git checkout main
2. git pull origin main
3. git add .
4. git commit --no-edit
5. git merge "your_brach"
6. git push origin main

 ## Fetch From Main:
1. git checkout "your_branch"
2. git fetch main
